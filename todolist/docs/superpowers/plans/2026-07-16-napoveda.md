# Příkaz `h` — nápověda (Help Command) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `h` command shows a modal man page of all commands; Enter returns to the task screen.

**Architecture:** New `TypPrikazu::Napoveda` in the parser; pure `vytiskniNapovedu(std::ostream&)` render function in `todolist.hpp`; `main()` handles the modal wait (clear screen, print, one discarded `getline`). Bottom command line gains ` | h`.

**Tech Stack:** C++17, existing plain-assert tests in `test_todolist.cpp`.

**Spec:** `docs/superpowers/specs/2026-07-16-napoveda-design.md`

**Build/test commands:**
```bash
g++ test_todolist.cpp -o test_todolist.exe -lsodium && ./test_todolist.exe
g++ todolist.cpp -o todolist.exe -lsodium
```
